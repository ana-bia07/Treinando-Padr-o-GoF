Aqui uso o Factory Method e Abstract Factory para criar uma pizzaria e uma hamburgueria.
Utilizo também os padrões Builder e Prototype para montar os sabores de pizza e os tipos de hambúrguer.

Por fim, aplico o padrão Singleton para gerenciar o pedido.

Sei que o Singleton não é o ideal para essa situação, já que cada cliente deveria ter seu próprio pedido individual. Pesquisando, descobri que existe uma variação do Singleton chamada Multiton, que seria muito mais eficaz na minha aplicação de teste, pois permite manter uma única instância por cliente.

Porém, como essa é uma atividade de aula, vou consultar meu orientador antes de implementar essa alteração, para verificar se está de acordo e como posso aplicá-la corretamente.
